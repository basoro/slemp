Ganti Konten

  • URL tujuan harus dapat diakses, jika tidak akan menghasilkan error 502
  • Secara default semua akses domain situs ini akan diteruskan ke server target, pastikan server target telah mengikat domain
  • Jika Anda adalah proxy pasif, silakan masukkan domain situs target pada kolom domain pengirim
  • Jika Anda tidak memerlukan fitur penggantian konten, biarkan saja kosong
  • Cache URL tertentu dapat dibersihkan melalui purge, contoh: http://test.com/purge/test.png
