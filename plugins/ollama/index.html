Tentang HTTP API

HTTP API memungkinkan akses ke Ollama melalui REST endpoints di port 5001. API ini menyediakan akses ke semua fungsi Ollama termasuk chat, model management, dan konfigurasi.

Available Endpoints

POST /api/plugins/ollama/generate

Generate response dari model

GET /api/plugins/ollama/models

Dapatkan daftar model yang tersedia

POST /api/plugins/ollama/models

Download model baru

DELETE /api/plugins/ollama/models

Hapus model
